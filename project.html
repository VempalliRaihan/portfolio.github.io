Title:Smart Agriculture using ESP8266

Project Discription:The aim of the project is smart agriculture and automatic irrigation system using ESP 8266 and humidity sensor with thingspeak,whenever the sensor pin detects a low quantity of humidity and temmperature in soil the water turns on automatically.and whenever the soil detects humidity then motor will automatically turns off.we can monitor all these via thingspeak server online from anywhere.
